# Machine Local
### Technologies needed
   - Go 1.14 or superior
   - Mysql 5.7

# Start Application
## Business Whatsapp
`go run business/whatsapp/main.go`

## Business Telegram
`go run business/telegram/main.go`

## Consumers Whatsapp
`go run consumers/whatsapp/main.go`

## Consumers Telegram
`go run consumers/telegram/main.go`

## Gateway Whatsapp
`go run gateway/main.go`

## Gateway Telegram
`go run telegram/main.go`

### Access
- Api: *http://localhost:3000*
